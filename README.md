# 🚀 Praxora

Praxora is a futuristic concept landing page exploring **Spatial Intelligence**, **Adaptive AI**, and **Human-Centered Computing** through immersive visuals, smooth animations, and a cinematic user experience.

> Designed as a concept interface inspired by next-generation AI systems and spatial computing.

---

## ✨ Features

- 🎨 Futuristic cyber-inspired UI
- 🌌 Smooth parallax scrolling effects
- 💎 Glassmorphism components
- ⚡ Interactive magnetic buttons
- 📱 Fully responsive layout
- 🎭 Scroll-based animations
- 🔷 Chrome-inspired visual identity
- ♿ Accessible navigation and semantic structure

---

## 🛠 Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

---

# 📂 Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/AARAYHARSHIT/Praxora.git
```

### 2. Navigate into the project

```bash
cd Praxora
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open in your browser

Visit:

```
http://localhost:3000
```
